➕ Новий абонемент

🔗 Публічні картки для батьків

Кожен клієнт отримує унікальне посилання. Батьки відкривають його і бачать стан абонементу в реальному часі. Коли ти відмічаєш заняття — картка оновлюється одразу (якщо налаштовано хмарне збереження).

☁️ Хмарне збереження (JSONBin.io)

Щоб картки батьків оновлювались в реальному часі, потрібне хмарне сховище. Зареєструйся безкоштовно на jsonbin.io → отримай API Key → вставь нижче.

📋 Як зареєструватись на JSONBin.io

  1. Відкрий jsonbin.io у новій вкладці
  2. Натисни Sign Up → зареєструйся безкоштовно
  3. Після входу натисни на своє ім'я → API Keys
  4. Скопіюй ключ (починається з $2a$10$)
  5. Вставь ключ вище і натисни Ініціалізувати хмару
  6. Готово — картки батьків тепер оновлюються!

🔄 Без хмари (локальний режим)

Якщо не хочеш реєструватись — все одно працює. Дані зберігаються в браузері. Картки батьків в цьому режимі не будуть оновлюватись автоматично, але ти можеш надіслати скріншот або скопійований текст через Telegram.

🔵 Google інтеграція (Таблиця + Календар)

Нові замовлення автоматично потраплятимуть у Google Таблицю (база клієнтів) і Google Календар (нагадування). Потребує одноразового налаштування.

📋 Як налаштувати (5 хвилин):
1. Відкрий console.cloud.google.com → створи проект
2. Увімкни Google Sheets API і Google Calendar API
3. Створи API Key → скопіюй сюди
4. Створи OAuth 2.0 Client ID (тип: Web) → додай https://igorzagorij5-dot.github.io як дозволений origin
5. Створи Google Таблицю → скопіюй ID з URL (між /d/ і /edit)
6. Натисни Авторизуватись → дай дозвіл → готово!
